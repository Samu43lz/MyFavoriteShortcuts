Atajos (Mis favoritos)

Para que me VS me de sugerencias:

(ctrl + space)

Aumentar el tamaño de la pantalla o disminuirla :

ctrl + "+" ó "-"

Seleccionar varios renglones a la vez para editarlos:

alt(en los renglones que deseas) + click

Encontrar código mediante palabras clave :

ctrl + f

Duplicar lineas:

shift + alt + (boton arriba ó abajo)

Comentar una parte de código ó bloque de código :

alt + shift + a

Comentar un bloque de código (2da opción) :

ctrl + "}"
